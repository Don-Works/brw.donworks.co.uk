# brw — website

The Next.js site for **[brw.donworks.co.uk](https://brw.donworks.co.uk/?utm_source=brw_site&utm_medium=readme&utm_campaign=donworks_oss)** — the landing page for `brw`, semantic browser control for agents.

The product itself lives in **[Don-Works/brw](https://github.com/Don-Works/brw)**. This repository is only the marketing/docs surface for the domain.
